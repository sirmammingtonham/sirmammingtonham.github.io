WorldWide Web/Ruin

For my English "Ruins" class during the last semester of high school (when I had too much time on my hands), we were tasked with exploring and writing about a ruin in some creative way. I thought it would be pretty cool to explore the "internet as a ruin." The "ruined" parts of the internet include old web design, code practices, and content.

I wanted to compare and contrast the "ruin internet" and the "modern web", so I created a website with two sides, one modeled after a modern website, complete with javascript animations, clean fonts, and compatibility across devices and browsers. The other side is modeled after a website you might find in late 1990s or early 2000s geocities, complete with gifs everywhere, background music, and even early 2000s word choice.

Each side of the website contains info about what aspects of the internet have changed over time in some interactive way, and links the old internet to the modern one. For example, the change from static websites designed to deliver information in the "Web 1.0" to dynamic web applications designed in part to collect information and a focus on mobile web design in the "Web 2.0".

I'm really proud of this project as it allowed me to learn a decent bit about web design and javascript. The modern site utilizes multiple "modern" javascript practices, including cloudflare rocket loader, floating buttons, jquery animations, cookie consent popups, adblock detection, modal popups, and hero images. The old site utilizes multiple html tags and css properties that one normally wouldn't see in modern web design.
